## Getting Started

Installation

1. Clone the respository :
   **https://github.com/raisalrasheed-gitlab/zyzk-website.git**
2. Navigate to the project directory :
   `$ cd client`
3. Install dependencies :
   `$ npm i or npm install`
4. Start the server :
   `$ npm run dev`
5. Open your browser and visit :
   **http://localhost:5173/**
